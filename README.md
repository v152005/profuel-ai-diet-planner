# 🥗 PROFUEL – AI Diet Planner

PROFUEL is a responsive web application that generates personalized diet recommendations based on a user's age, height, weight, and fitness goals. It helps users make healthier food choices through simple calculations and nutritional guidance.

## 🚀 Live Demo
https://v152005.github.io/profuel-ai-diet-planner/

## ✨ Features
- BMI Calculator
- Daily Calorie Estimation
- Personalized Diet Recommendations
- Weight Loss, Maintenance & Muscle Gain Plans
- High-Protein Food Suggestions
- Responsive User Interface

## 🛠️ Tech Stack
